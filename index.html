Lua Commands for Game Development

Essential Lua functions and techniques for game development

Tables and Data Structures

table.insert()
Inserts a value into a table at a specified position or at the end if no position is specified.
local fruits = {"apple", "banana"} table.insert(fruits, "orange") table.insert(fruits, 2, "grape")

How to use:

  1. Specify the table as the first argument.
  2. Optionally specify the position as the second argument.
  3. Provide the value to insert as the last argument.
table.remove()
Removes an element from a table at a specified position.
local fruits = {"apple", "banana", "orange"} local removed = table.remove(fruits, 2)

How to use:

  1. Specify the table as the first argument.
  2. Optionally specify the position as the second argument (defaults to last element).
  3. Returns the removed element.

String Manipulation

string.format()
Returns a formatted string based on a format string and additional arguments.
local name = "Player" local score = 1000 local message = string.format("%s scored %d points!", name, score)

How to use:

  1. Provide a format string as the first argument.
  2. Use %s for strings, %d for integers, %f for floats, etc.
  3. Provide additional arguments to fill the placeholders.
string.sub()
Returns a substring of a given string.
local str = "Hello, World!" local sub = string.sub(str, 1, 5) -- Returns "Hello"

How to use:

  1. Provide the main string as the first argument.
  2. Specify the start index as the second argument.
  3. Optionally specify the end index as the third argument.

Math and Random Number Generation

math.random()
Generates a random number within a specified range.
local randomNumber = math.random() -- Returns a number between 0 and 1 local diceRoll = math.random(1, 6) -- Returns a number between 1 and 6

How to use:

  1. Call without arguments to get a float between 0 and 1.
  2. Provide one argument for an integer between 1 and that number.
  3. Provide two arguments for an integer within that range (inclusive).
math.floor() and math.ceil()
Rounds a number down (floor) or up (ceil) to the nearest integer.
local x = 3.7 local floorX = math.floor(x) -- Returns 3 local ceilX = math.ceil(x) -- Returns 4

How to use:

  1. Pass a number as an argument to either function.
  2. Use floor() to round down, ceil() to round up.
  3. Useful for pixel-perfect positioning in games.

File I/O

io.open()
Opens a file for reading or writing.
local file = io.open("savegame.txt", "w") file:write("Player Score: 1000") file:close()

How to use:

  1. Specify the filename as the first argument.
  2. Provide the mode as the second argument ("r" for read, "w" for write, "a" for append).
  3. Use methods like write() or read() on the returned file object.
  4. Always close the file when done.

Time and Date

os.time()
Returns the current time as the number of seconds since the epoch.
local currentTime = os.time() local futureTime = os.time({year=2025, month=1, day=1})

How to use:

  1. Call without arguments to get the current time.
  2. Pass a table to get the time for a specific date.
  3. Useful for calculating time differences or setting future events.

Coroutines

coroutine.create() and coroutine.resume()
Creates a new coroutine and resumes its execution.
local co = coroutine.create(function() for i = 1, 3 do print("Coroutine step", i) coroutine.yield() end end) coroutine.resume(co) -- Prints "Coroutine step 1" coroutine.resume(co) -- Prints "Coroutine step 2"

How to use:

  1. Create a coroutine with coroutine.create(), passing a function.
  2. Use coroutine.resume() to start or continue the coroutine's execution.
  3. Use coroutine.yield() within the coroutine to pause execution.
  4. Useful for implementing game state machines or managing complex sequences.

Metatables and Metamethods

setmetatable() and getmetatable()
Sets or gets the metatable of a table, allowing for custom behavior.
local t = {} local mt = { __index = function(t, k) return "Key " .. k .. " not found!" end } setmetatable(t, mt) print(t.nonexistent) -- Prints "Key nonexistent not found!"

How to use:

  1. Create a metatable with desired metamethods (__index, __newindex, etc.).
  2. Use setmetatable() to associate the metatable with a table.
  3. Use getmetatable() to retrieve the metatable of an object.
  4. Powerful for implementing object-oriented programming in Lua.
